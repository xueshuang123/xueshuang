数据分析技术

进行中 · 大二下学期

从“会用工具”过渡到“理解原理并解决复杂业务问题”。本课程专为已掌握Python基础和采集技术的同学设计，重点在于思维训练、特征工程和模型应用。

2026年春季学期 6个模块 · 10个实战项目 Python + Pandas + Scikit-learn

📖 课程概述

先修要求

  • Python基础语法（变量、循环、函数）
  • Pandas数据处理基础
  • 数据采集与处理课程
  • 商务数据分析基础

学习目标

  • 掌握结构化思维拆解业务问题
  • 理解特征工程的原理与方法
  • 能够独立完成AB测试设计与分析
  • 入门机器学习并应用于商业场景

考核方式

  • 平时作业 (30%)：6次编程作业
  • 课堂参与 (10%)：讨论与分享
  • 综合大作业 (60%)：电商全链路分析

📋 课程大纲

模块 1

数据分析思维与进阶流程

内容：回顾CRISP-DM流程，引入结构化思维（MECE原则、逻辑树、金字塔原理）拆解业务问题。对比Excel与Python分析流程的优劣。

目标：培养面对陌生数据时有清晰的切入思路。

实战项目：业务问题拆解练习
模块 2

特征工程与数据处理进阶

内容：正则表达式高阶清洗、标准化(StandardScaler)与归一化(MinMaxScaler)的原理与场景、PCA主成分分析可视化。

实战项目：《信贷用户违约特征筛选》
模块 3

统计分析与AB测试实践

内容：概率分布、中心极限定理、置信区间、假设检验(T检验/卡方检验)。AB测试全流程：样本量计算、显著性判断、业务决策。

实战项目：《电商落地页AB测试分析》
模块 4

机器学习入门(上) - 回归

内容：线性回归、逻辑回归的原理推导与SKlearn实现。模型评估：R²、混淆矩阵、ROC/AUC曲线。

实战项目：《员工离职预测模型》
模块 5

机器学习入门(下) - 聚类与关联规则

内容：K-Means聚类原理与手肘法选K值、RFM用户分层模型进阶、Apriori算法与购物篮分析。

实战项目：《超市会员RFM分层》+《商品关联规则挖掘》
模块 6

综合大作业

内容：整合本学期知识，完成从数据清洗、探索性分析、特征工程到用户分群的完整报告。

实战项目：《电商全链路数据分析》

💻 10个梯度实战项目

从基础到进阶，每个项目都配有业务场景、关键思维点拨和完整数据集。

入门

项目1：咖啡店销售报表

掌握 groupby 聚合与排序，回答“哪一天卖得最好？”

入门

项目2：电商用户行为漏斗

计算浏览→加购→下单→支付转化率，定位流失环节。

入门

项目3：会员RFM分层

计算最近消费(R)和频次(F)，给用户打标签。

进阶

项目4：房源数据清洗

处理空值、异常值、格式不一致的脏数据。

进阶

项目5：客服聊天情绪分析

中文分词、词云图、简单情感字典匹配。

进阶

项目6：AB测试效果评估

箱线图解读、异常值识别、统计显著性判断。

进阶

项目7：员工离职因素分析

相关性矩阵、热力图绘制、多因素关联解读。

高阶

项目8：用户消费聚类画像

数据标准化、KMeans聚类、手肘法选K值。

高阶

项目9：购物篮关联规则

One-Hot编码、Apriori算法、支持度与提升度。

高阶

项目10：月度销量预测

时间序列索引、移动平均、趋势与季节性分解。
